# OnlineStore

A simple online store to add and order products.

## Project Overview
OnlineStore is an e-commerce web application with two main roles: **Admin** and **Consumer**.

### Admin Features
- Add new products
- Edit existing products
- Delete products
- Add product categories
- Edit categories
- Delete categories

### Consumer Features
- Browse products
- Add products to cart
- Edit cart items
- Remove items from cart
- Place orders
- View details of previous orders
